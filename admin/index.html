⚠️ Configuration requise

Netlify Identity n'est pas activé sur votre site. Suivez ces étapes :

  1. Allez sur vprrtestblog.netlify.app
  2. Cliquez sur "Site settings"
  3. Dans le menu de gauche, cliquez "Identity"
  4. Cliquez "Enable Identity"
  5. Puis "Services" → "Enable Git Gateway"

Après activation, rechargez cette page.
